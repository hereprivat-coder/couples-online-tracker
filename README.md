# couples-online-tracker

Собирает раз в 15 минут (через GitHub Actions) агрегированное число моделей
категории "пара" (Couple\*), сейчас в эфире на BongaCams — публичный
affiliate API, без авторизации, без каких-либо личных данных.

Лог — `couples_online_log.csv` (ts_utc, couples_online, total_online).
Используется личным дашбордом для анализа "в какие часы больше/меньше пар
в эфире" и сверки с собственным заработком.
